# 🚀 Simple Minify

## 📌 Overview
A modern and optimized web application focused on improving website performance and delivering smooth user interactions.  
This project was built as part of an advanced front-end internship task to demonstrate performance optimization techniques and advanced animations while maintaining accessibility and responsiveness.

---

## ✨ Features
- Optimized images using modern formats (WebP)
- Lazy loading for faster initial page load
- Minified and bundled CSS / JavaScript files
- Reduced render-blocking resources
- Improved Google Lighthouse performance metrics
- Browser caching strategies for faster repeat visits
- Smooth and performant animations
- Interactive hover and scroll-based effects
- Responsive design across all devices
- Clean and modern UI

---

## 🎬 Advanced Animations
- Smooth transitions and motion effects
- Hover interactions for better engagement
- Scroll-triggered animations
- Optimized animations with minimal performance impact
- Accessibility-friendly motion behavior

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- GSAP / CSS Animations
- Google Lighthouse
- Vercel Deployment
